  • Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'. (But not those ones):
    • Control flow is the order in which individual programming codes are executed or evaluated. For instance. Checking your friends list on social media. For every friend there's a individual variable number. You could program this by using control flow. But it would be a repeatative process. Whereas, loops are enabled to shorten programs/ applications.

    Reference:

    Control flow:

    show_friend($user->friends[0]);
    show_friend($user->friends[1]);
    show_friend($user->friends[2]);

    Loops:

    foreach(var $friend in $user->friends) { show_friend($friend); }

  • Explain the difference between accessing data from arrays and objects:
    • Objects are used to represent a “thing” in your code. Properties in objects can be accessed, added, changed, and removed by using either dot or bracket notations. Whereas, arrays use zero-based indexing. The first item in an array has an index of 0, the second item an index of 1, and so on.
